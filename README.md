# XJTU-Engineering-Optics

> 西安交通大学仪器类专业《工程光学》学习资料库

## 📖 写在前面

本仓库是西安交通大学仪器类专业专属传奇课程《工程光学》的学习资料。本仓库目前整理了课程PPT、往年题、学习笔记等资料，希望让大家高效地掌握本课程的内容，同时欢迎各位补充。

## 📂 仓库结构

- /PPT: 课程课件
- /Exams: 历年真题
- /Notes: 学习笔记
- /Answers: 习题答案
- /Books：推荐书单

## 🤝 贡献与反馈

本仓库的长期维护需要大家共同参与！如果你有以下资料，欢迎提交Issue

1. 最新年份的期末试题（回忆版亦可）；
2. 优质的个人复习笔记。

### 如何贡献

1. 在仓库页面点击 Issues 标签页
2. 点击 New Issue
3. 描述资料介绍
4. 拖拽或上传PDF、Word等文件
